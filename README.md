
# iEvent 

A userfriendly site that allows users to post public events so that others can join and find all relevant details. 





## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

```bash
  python3 -m pipenv install flask
```

```bash
  pipenv shell
```

```bash
 python3 server.py
```

You will need to follow the ERD and setup a local db on your system.
